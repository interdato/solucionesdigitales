# Interdato · Soluciones digitales para logística, transporte e industria

Repositorio para GitHub Pages con un portafolio interactivo de servicios de TI para vender en expo o reuniones comerciales.

## URL esperada

https://interdato.github.io/solucionesdigitales/

## Qué incluye

- Landing profesional con branding de Interdato.
- Demos interactivas de dashboard, base de datos, trazabilidad QR, IA con cámaras e integraciones.
- Videos explicados para prospectos:
  - Dashboard operativo.
  - IA conceptual con cámaras.
  - Detección de EPP.
  - Inventario en racks.
  - Postura y ergonomía.
  - Validación de EPP en zonas operativas.
- Calculadora de ahorro mensual estimado.
- Presentación PDF y PPTX.
- Archivos de ejemplo SQL, CSV, JSON y flujo de automatización.

## Estructura

```text
index.html
styles.css
script.js
.nojekyll
assets/
  docs/
  images/
  videos/
  data/
```

## Edición rápida

- Textos de la página: editar `index.html`.
- Estilos visuales: editar `styles.css`.
- Interacciones: editar `script.js`.
- Videos: reemplazar archivos en `assets/videos/` manteniendo el mismo nombre.
- Presentación: reemplazar PDF/PPTX en `assets/docs/` manteniendo el mismo nombre.
