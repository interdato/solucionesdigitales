# Interdato · Soluciones digitales para logística, transporte e industria

Repositorio para GitHub Pages con un portafolio interactivo de soluciones digitales orientado a empresas de logística, transporte, almacén e industria.

## Contenido

- Landing comercial con branding de Interdato.
- Demos interactivas de dashboard, base de datos, trazabilidad QR, detección de zona de riesgo e integraciones.
- Videos demostrativos de dashboards, IA visual, EPP, inventario en racks, postura/ergonomía y validación de EPP.
- Calculadora de ahorro potencial.
- Archivos de ejemplo: SQL, CSV, JSON y flujo de automatización.
- Presentación PDF y PPTX.
