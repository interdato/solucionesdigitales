# Flujo de automatización propuesto

1. El operador registra una orden, evidencia o escaneo QR.
2. El sistema valida campos obligatorios, estado operativo y reglas de negocio.
3. Si existe una incidencia, se genera una alerta para operaciones.
4. El evento queda guardado en una bitácora auditable.
5. El dashboard se actualiza con indicadores de cumplimiento, retrasos y productividad.
6. Cuando aplica, se envía notificación por correo, WhatsApp o integración con ERP/TMS/WMS.

Este flujo puede aplicarse a rutas, patios, unidades, almacenes, mantenimientos, activos, inventario o entregas.
